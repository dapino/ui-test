# ui-test
This is a react app
___

## Installation
Install dependencies

```npm i```

Install json-serve to create an fake API

``` npm i json-server -g ```

___

## Usage
 Run app in development mode
 
 ```npm start```
 
 Build dist app
 
 ```npm run build```
 
 Start fake API
 ```json-server ./src/data/characters.json ```
